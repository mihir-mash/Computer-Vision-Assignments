# Computer Vision Assignments

This repository contains a collection of hands-on assignments covering fundamental and advanced concepts in Computer Vision. Each assignment focuses on a specific technique or application, implemented using Python and Jupyter Notebooks.

## Repository Structure

The repository includes the following assignments:

* **Assignment 1**

  * `dress_illusion_analysis.ipynb`
    Analysis of the famous dress illusion using image processing techniques.
  * `morphological_operations.ipynb`
    Implementation of basic morphological operations such as erosion, dilation, opening, closing, etc.

* **Assignment 2**

  * `frequency_domain_filtering.ipynb`
    Image filtering in the frequency domain using Fourier Transform techniques.

* **Assignment 3**

  * `image_segmentation_techniques.ipynb`
    Exploration of various image segmentation methods.

* **Assignment 4**

  * `object_detection_predictions.pdf`
    Report/document showcasing object detection results and analysis.

* **Assignment 5**

  * `yolov8_object_detection.ipynb`
    Object detection using the YOLOv8 model.

* **Assignment 6**

  * `Image_Colorization_With_GANs.ipynb`
    Image colorization using Generative Adversarial Networks (GANs).

* **Assignment 9**

  * `human_pose_estimation_with_movenet.ipynb`
    Human pose estimation using MoveNet.

## Technologies Used

* Python
* OpenCV
* NumPy
* Matplotlib
* TensorFlow / PyTorch (depending on assignment)
* YOLOv8 (Ultralytics)
* GAN architectures

## Objectives

* Understand core image processing techniques
* Implement classical and deep learning-based computer vision models
* Gain practical exposure to real-world CV tasks such as detection, segmentation, and pose estimation
* Develop intuition for model performance and limitations

## Notes

* Each notebook is self-contained with explanations and code.
* Some assignments may require GPU support for efficient execution.
* Ensure all required libraries are installed before running.
